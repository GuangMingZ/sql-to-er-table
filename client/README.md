# 云知知识库 前端项目

## 技术栈

- 核心框架 ：React18
- 构建工具 ：Vite
- 状态管理 ：Zustand
- 样式解决方案 ：Less
- 类型系统 ：TypeScript
- 包管理器：npm

## 环境要求

node.js >= 18

## 启动项目

### 安装依赖

```bash
# 安装依赖
npm install

```

### 添加配置文件

**`.env` 文件请找 @broxiang @guangmzhai 获取，因包含安全信息，不能上传到 git 仓库里**
`.env` 请放到根目录下

**`config.json` 文件请找 @broxiang @guangmzhai 获取，因包含安全信息，不能上传到 git 仓库里**
`config.json` 请放到 `.local` 目录下

### 启动项目

```bash
npm run dev
```
